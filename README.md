# PeerLink
PeerLink — это разрабатываемый децентрализованный P2P-мессенджер на C++, создаваемый как основа для массово используемой распределённой системы связи без обязательного центрального сервера. Текущая версия находится в активной разработке и предназначена для раннего тестирования, отладки сетевой архитектуры и постепенного расширения функциональности.
